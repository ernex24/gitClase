Hola esto es un repositorio donde tenemos que

  • Inicializar un repositorio

  • • Inicializamos el repositorio

  • git init
  • git add .
  • git commit -m "primer commit"

  • • Enlazar con un repositorio remoto

  • git remote add origin https://github.com/ernex24/gitClase.git
  • git push -u origin master

  • • Obtener un repositorio remoto

  • git clone https://github.com/ernex24/gitClase.git

  • • Permisos de acceso a un repositorio remoto

  • git config --global user.name "Ernesto Perez"
  • git config --global user.email ernex24@gmail.com
  • git config --global core.editor vim
  • git config --global merge.tool vimdiff

  • • Listando los últimos cambios del repositorio

  • git status

  • • Obtener los últimos cambios del repositorio remoto

  • git pull

  • • Examinar el historial de cambios de un archivo

  • git log

  • • Crear una nueva rama en el repositorio remoto

  • • Diferencias entre 2 ramas de nuestro proyecto

  • git diff master..nuevaRama

  • • Manejar etiquetas (tags)

  • git tag 1.0.0
  • git push tag

  • • Crear una rama local en el repositorio remoto

  • git checkout -b nuevaRama
  • git push origin nuevaRama

  • • ¿Quién hizo qué? Uso de: git blame

  • git blame --line --porcelain index.html
